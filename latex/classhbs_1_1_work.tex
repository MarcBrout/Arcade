\subsection{Detailed Description}
\hyperlink{classhbs_1_1_work}{Work} is an interface that allow classes to be threaded by the \hyperlink{classhbs_1_1_workers}{Workers} class. 

The \hyperlink{classhbs_1_1_work}{Work} class is an interface that will allow the class which inherit to be used by the thread pool \hyperlink{classhbs_1_1_workers}{hbs\-::\-Workers}. The design is really simple\-: you the work is registered in the \hyperlink{classhbs_1_1_workers}{Workers} class with an integer that will be sent to the operator() function. The function will return another integer that will be stored inside the \hyperlink{classhbs_1_1_workers}{Workers} class until the Get\-Message function is called.
