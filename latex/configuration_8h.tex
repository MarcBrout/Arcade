\subsection{Detailed Description}
The compat I\-N\-I file contains functions that were used by previous version of the Lib\-Lapin to handle I\-N\-I file.

A I\-N\-I file data structure is defined like this\-: \#here, we are in D\-E\-F\-A\-U\-L\-T\-\_\-\-S\-C\-O\-P\-E \mbox{[}scopename\mbox{]} \#here, we are in scopename fieldname=index0,index1,index2,index3 \#this is the field fieldname in scopename

These symbols and functions are obsolete and should not be used anymore. Insteadk use the new bunny\-\_\-configuration A\-P\-I that can be found inside \hyperlink{configuration_8h}{configuration.\-h}. These functions are still present for backward compatibility
