typedef struct s_bunny_accurate_area t_bunny_accurate_area

The t_bunny_accurate_area structure describes an area in a 2D space. It may be used to check collision between different areas.

typedef struct s_bunny_accurate_position t_bunny_accurate_position

The t_bunny_accurate_position represents a position in a 2D space. It is useful to describe elements in a virtual and smooth space. Using t_bunny_accurate_position allows you to use float or double to represent speed or any kind of physical concept that need to be precise in order to not being restrictive. For example, having a t_bunny_accurate_position element on screen with double for x and y speed allows it to move in every direction at any speed.

typedef t_bunny_accurate_position t_bunny_accurate_size

The t_bunny_accurate_size represents the size of an element in a 2D space with precision.

typedef struct s_bunny_area t_bunny_area

The t_bunny_area structure describes an area in a 2D space. It may be used to check collision with fixed elements like tiles, that are often int-accurate only.

typedef struct s_bunny_position t_bunny_position

The t_bunny_position structure represents a position in a 2D space. It is useful to access to memory space or tiles, but not to register position of elements in a virtual and smooth space. Use t_bunny_accurate_position instead.

typedef t_bunny_position t_bunny_size

The t_bunny_size structure represents the size of an element in a 2D space. It is useful to describe size of elements in memory or low resolution element size. If you wish more precision, use t_bunny_accurate_size instead.
