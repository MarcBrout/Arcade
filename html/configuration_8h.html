Detailed Description

The compat INI file contains functions that were used by previous version of the LibLapin to handle INI file.

A INI file data structure is defined like this: #here, we are in DEFAULT_SCOPE [scopename] #here, we are in scopename fieldname=index0,index1,index2,index3 #this is the field fieldname in scopename

These symbols and functions are obsolete and should not be used anymore. Insteadk use the new bunny_configuration API that can be found inside configuration.h. These functions are still present for backward compatibility
