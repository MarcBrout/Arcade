Detailed Description

Workers is a thread pool made to work with the Work class.

The Workers class allows the user to ask for a specific amount of thread. The amount of thread can be changed. You also can watch the load balancy and act in consequence.

You can add works in the Workers class, one at time, or several in one single call. Getting returned message is easy thanks to the GetMessage function.

You can watch how many threads are running, so you can fork instead of adding new threads if you think you are full.
