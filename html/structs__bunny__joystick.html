Detailed Description

The t_bunny_joystick structure contains all informations about a joystick slot. The connected attribute indicates if the slot is used. The name attribute is the name of the joystick The vendor attribute is the ID of the vendor The product attribute is the ID of the product. The nb_button attribute is how many buttons there is The axis attribute is how many axis there is: horizontal, vertical, etc. See the enum.h file for more informations.
